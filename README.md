# adrianhall.github.io

**This repository is retired.** My blog has moved to
[blog.adrianhall.uk](https://blog.adrianhall.uk) (rebuilt on
Astro/Cloudflare — see `MIGRATION_PLAN.md`).

The `main` branch is a frozen historical record of the old Jekyll source
that used to build this site. It is no longer built or deployed.

`adrianhall.github.io` now serves a small client-side redirect to the new
site for every old URL. That redirect's source lives on the
[`site`](../../tree/site) branch, published via GitHub Pages' "Deploy from
a branch" setting — see `M6_PLAN.md` for details.
